## Roles

### User Management
#### Features
- Login
- Logout
- Password Reset
- Role-based Access Control
- User Profiles
- Account Activation
- User Status Management

#### Roles
- Admin
- Professor
- Student

### Administrator
Responsible for managing the entire LMS ecosystem.

#### Permissions
- User Management
- Course Management
- KPI Configuration
- Dashboard Monitoring
- Analytics
- Reports
- Alert Management
- Question Bank Management
- System Configuration
- Audit Logs

### Professor
Responsible for creating and managing academic content.

### Course Management

#### Professors shall be able to:
- Create Courses
- Edit Courses
- Archive Courses
- Publish Courses
- Assign Students
- Categorize Courses

#### Admin shall:
- Approve Courses
- Archive Courses
- Monitor Courses

#### Students shall:
- Access Assigned Courses
- View Course Outline

#### Permissions
- Create Courses
- Create Lectures
- Upload Videos
- Upload PDFs
- Upload Books
- Upload Notes
- Upload Worksheets
- Create Assignments
- Create Quizzes
- Select Questions from Question Bank
- Monitor Student Progress
- Review Analytics
- Receive KPI Notifications
